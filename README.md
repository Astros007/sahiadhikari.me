# Sahisnu Adhikari Portfolio

A static multi-page portfolio website prepared for GitHub Pages.

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload **everything inside this folder**, including the `.github` folder.
3. Keep the default branch named `main`.
4. Open **Settings → Pages**.
5. Under **Build and deployment**, set **Source** to **GitHub Actions**.
6. Open the **Actions** tab and wait for **Deploy portfolio to GitHub Pages** to finish.

The live address will be shown in the deployment summary and in **Settings → Pages**.

## Main files

- `index.html` — home page
- `research.html` — research
- `projects.html` — projects
- `clubs.html` — clubs and leadership
- `awards.html` — awards
- `experience.html` — experience
- `resume.html` — resume page
- `contact.html` — contact page

All HTML pages and PDFs must remain together at the repository root because the site uses relative links.
